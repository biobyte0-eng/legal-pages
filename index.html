BioBytee Analytics Extractor

App Home Page (OAuth verification)

What this app does

BioBytee Analytics Extractor is a personal tool that helps a YouTube creator export read-only channel and video analytics for their own content into CSV/Parquet files for analysis and model training.

How it works

  • Uses Google OAuth to request access to the YouTube Analytics API and YouTube Data API.
  • Fetches analytics for videos on the authenticated user’s own channel.
  • Exports metrics locally (for example, to Google Drive / a local file store) for the user’s analysis.

Data & permissions

This app requests only the permissions needed to read analytics and channel/video metadata. Data is used only for the functionality described above and as disclosed in the Privacy Policy.
